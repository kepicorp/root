## Marquise

- I want to have a button to march where I can select up to two sources clearing and then move the amount of warrior I want.

Here is how March works:
March Action (Marquise de Cat)
A single March action actually gives you two separate Moves. Each Move follows the standard rules:

Pick an origin clearing where you have rule
Move any number of warriors and/or buildings to one adjacent clearing

So with one March action you get 2 moves, and each of those moves can:

Come from different origin clearings (or the same one twice)
Move any number of warriors each time

Example:

Move 3 warriors from clearing A to clearing B
Then move 2 warriors from clearing C to clearing D

All within a single March action.
